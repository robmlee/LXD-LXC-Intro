## 一、建立 n8n 專用 Container
- **1. 建立 Container**
  ```bash=
  ##
  lxc copy ub2204 n8n
  ```
  
- **2. 啟動 Container**
  ```bash=
  ##
  lxc start n8n
  ```

- **3. 登入 Container**
  ```bash=
  ##
  lxc exec n8n -- sudo --login --user ubuntu
  ```
